# Resource-Finder
Make sure you have necessary libraries and are running python3:
pip3 install Flask
pip3 install beautifulsoup4

If running on Linux:

	export FLASK_APP=index.py

	python3 -m flask run

If running on Windows:

	set FLASK_APP=index.py

	python3 -m flask run

And go to: http://127.0.0.1:5000:/index.html
